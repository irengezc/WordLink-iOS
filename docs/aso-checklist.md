# ASO Checklist

## Goal

Improve App Store discoverability and conversion. For a solo founder with no ad
budget, organic App Store search is the main acquisition channel.

ASO is continuous, not a one-time launch task.

## Positioning

Lead with the real product:

- Word puzzle game.
- Chain-solving mechanic.
- English vocabulary / ESL fluency benefit.

Avoid making the store page feel like a generic learning app. The hook is a game
that also improves vocabulary.

## App Store Assets

Icon:

- Distinct at small sizes.
- Reads as word puzzle / link / chain without tiny text.
- Avoid clutter and low-contrast details.

Screenshots:

- Show the actual game mechanic within about 2 seconds.
- Include one screenshot focused on the chain-solving interaction.
- Include one screenshot focused on explanations / learning value.
- Include one screenshot that makes difficulty or progression clear.
- Use captions that are short and benefit-led.

Preview video or GIF, if added:

- Show entering a guess.
- Show a correct connection.
- Show the phrase explanation.
- Do not spend the first seconds on branding only.

## Listing Copy

Title/subtitle:

- Include word puzzle language.
- Preserve the ESL / vocabulary angle where natural.

Keyword themes:

- word puzzle
- word game
- vocabulary
- English learning
- ESL
- compound words
- idioms
- daily puzzle, once daily challenge ships

Description:

- Explain the 9-word chain in plain language.
- Mention 8 hidden connections.
- Mention the three difficulty levels.
- Emphasize learning through phrase explanations.
- Avoid promising features not yet shipped, such as battle mode or flashcards.

## Ratings Plan

Goal: secure the first 10-20 ratings from happy users.

Prompt timing:

- After completing a level/chain.
- Prefer after a good outcome, not after a wrong guess or failed load.
- Do not interrupt active gameplay.

Implementation note:

- Use Apple's in-app review prompt API where appropriate.
- Add a cooldown so the prompt does not feel repetitive.

## Ongoing Loop

Review every release:

- Which keywords are in the current listing?
- Which screenshot best communicates the mechanic?
- Are reviews mentioning confusion, delight, or learning value?
- Did daily challenge or monetization change what screenshots should show?

## Acceptance Criteria

- Store assets show the real game, not abstract marketing.
- ESL/value positioning is visible without hiding the game mechanic.
- Rating prompt appears after a positive completion moment.
- Copy does not claim deferred features.
- ASO changes are revisited after meaningful product changes.
